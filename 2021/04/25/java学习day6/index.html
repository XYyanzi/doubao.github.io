字符串

字符串的特点:
1、字符串的内容永不可变,是一个常量
2、正是因为字符串不可改变，所以字符串是可以共享使用的。
3、字符串效果上相当远char[]字符数组，但是底层原理是byte[]字节数组

创建字符串的方式

三种构造方法，一种直接创建

  • public String(): 创建一个空白字符，不含任何内容
  • public String(char[] array): 根据字符数组的内容来创建字符串
  • public String(byte[] array): 根据字节数组的内容来创建字符串
  • String str = “hello”; 右边直接用双引号

注意:直接写上双引号，就是字符串对象

字符串常量池: 程序中直接写双引号创建的字符串，就在字符串常量池中。 【堆内存中保存, 实际保存的是字节数组的地址, new 创建的字符串不再常量池中。
==判断: 对于基本数据类型，比较的是值，对于引用类型，比较的是地址值

equals()方法

如果要判断两个引用对象的值是否相等，可以使用equals方法:
public boolean equals(Object obj)
注意事项:
1、equals方法具有对称性，a.equals(b) 和b.equals(a)是一样的
2、如果比较的是一个常量和一个变量，推荐将常量写在前面 “q”.equals(b),防止变量的值是null的情况下报空指针错误.
3、equalsIgnoreCase() 方法可以忽略英文大小写

static关键字

被static关键字修饰的成员变量或成员方法，不属于对象自己，而是属于类，本类对象共享。

注意事项:
1、静态不能直接访问非静态
原因：在内存当中【先】有静态内容，【后】有的非静态
2、静态方法中不能使用this
原因：this是只当前对象，但静态方法不需要对象
3、静态优先于非静态，因此静态代码块比构造函数先执行

静态代码块的典型用途: 用来一次性的对静态变量进行赋值

静态变量和静态方法在内存中是在方法区的静态区中。
