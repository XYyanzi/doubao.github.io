final关键字

final关键字代表最终的、不可改变的。

可以修饰一个类: 被修饰的类不能有任何子类
可以修饰一个方法: 被修饰的方法不能被子类覆盖重写
可以修饰一个局部变量: 被修饰的局部变量的值不能进行更改，一次赋值终生不变
可以修饰一个成员变量: 被修饰的成员变量也不能改变，由于成员变量具有默认值，如果用了final修饰必须手动赋值。

Object类和常用API

java.lang.object类是java语言中的根类，即所有类的父类。如果一个类没有特别指定的父类，那么默认继承Object类。

toString方法

看一个类是否重写的toString方法，直接打印类对象的名字即可。没有重写toString方法打印的就是地址值。

equals方法

equals方法源码逻辑: this == obj
对于基本数据类型,比较的是值
对于引用数据类型,比较的是地址值
如果引用数据类型需要比较属性值，需要重写equals方法
