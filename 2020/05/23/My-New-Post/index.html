创建我的blog——Hexo+github

发表于 2020-05-23 更新于 2020-07-05 分类于 Hexo

心血来潮想要像其他专业技术人员一样有个自己的博客，记录和分享自己的各种采坑过程，那么就从创建博客开始吧~
基于Mac OS

安装基础环境

官网党可以参考官网进行安装, 上面有详细的步骤,一手资料最靠谱。对于想偷懒的小伙伴也可以直接看参考我的安装步骤

  1. 安装Node.js
    用来生成静态页面,具体安装方式参考Node.js官网,下载mac pkg安装包一路安装即可。
    执行node -v 检查是否安装成功
  2. 安装Git
    用来将本地Hexo内容提交到Github上。这个可以跳过，基本大家都有git了。没有的话可以通过homebrew安装或者装个Xcode

添加ssh key到Github避免每次都需要输入用户名和密码

如果本地没有配置github账号，之后每次提交可能都需要输入用户名和密码，使用起来很不方便，因此需要设置一下。如果你的电脑之前已经配置过ssh key，那么这一步可以跳过。

  1. 检查本地电脑是否有ssh key, 进入~/.ssh目录下查看是否存在文件id_rsa.pub和id_dsa.pub，如果已经存在，可直接进行步骤2，将ssh key添加到github上，如果不存在，则输入下面命令生成ssh key，其中的邮箱地址换成你自己的github邮箱

  2. 将~/.ssh目录下的id_rsa.pub文件中的内容复制到github的ssh key中。
    进入Github –> Settings –> SSH keys –> add SSH key添加

创建第一篇文章

blog搭建好了，之后就可以愉快的编写文章了

  1. 创建新文章
    • 进入blog项目目录下，执行hexo new My New Post,其中My New Post就是你的文章名称，执行之后，会source/_post目录下生成一个My-New-Post.md的文件，这时就可以打开这个文件愉快的写你的第一个博客了。
  2. 编辑后发布
    • 编辑完文章之后，保存，然后回到blog目录下，执行命令
      hexo g 生成静态页面
      hexo s 进行本地预览
      hexo d 部署发布到远程

到此为止属于你自己的blog就算搭建成功啦~下面再介绍一些自定义设置让你的blog更丰富。

设置NexT主题

blog搭建好之后，可以选择自己喜欢的主题来美化自己的blog了。Hexo的官网上提供了大量的主题，可以从中选择自己喜欢的进行配置。我是看到知乎推荐第一的主题是Next并且官方文档比较齐全，因此选择了Next主题，下面来看看怎么配置主题吧。使用指导

  1. 安装主题教程

    • 进入到blog的根目录下，该目录下会有themes目录

    • 在根目录下创建新目录

      这种安装方式没有办法通过git更新最新版本

  2. 将blog的主题设置为next

    • 打开根目录下的_config.yml,将theme值改为next
      theme: next

  3. 检查预览，执行hexo clean && hexo g -d，然后执行hexo -s生成本地连接查看主题效果
